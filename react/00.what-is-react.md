# React 란?

*A JavaScript library for building user interfaces*

UI 개발을 위한 Javascript Library.

```javascript
ReactDOM.render(
  <h1>Hello, world!</h1>,
  document.getElementById('root')
);
```
대충 이런 코드를 사용한다.

React는 Javascript 라이브러리이기 때문에 Javascript 기본 지식이 요구된다.
