# 비트 연산자

```
Operator    Description             Example
&           AND 연산                6 & 3 = 2, 110 & 010 = 010
|           OR 연산                 6 | 3 = 6, 110 |010 = 110
^           XOR 연산                6 ^ 3 = 5, 110 ^ 010 = 101
~           보수 연산               ~6 = -7, ~110  = -111
<<          왼쪽 쉬프트 연산        6 << 2 = 24, 110 << 2 = 11000
>>          오른쪽 쉬프트 연산      6 >> 2 = 1, 110 >> 2 = 001
```
